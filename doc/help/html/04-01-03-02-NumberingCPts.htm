Numbering Control Points

The ‘cvt’ table and other text-based windows refer to control points by numbers that come from the glyph outline description. To determine the number of a point, you can display these numbers, which Visual TrueType uses to keep track of the points. Numbering continues clockwise on the outer contours of a glyph and counter-clockwise on the inner contours.

Each time you add or delete a point, Visual TrueType renumbers all the points and updates the point numbers in the text-based tables. These numbers are useful for advanced users who want to edit source data right in a text-based window. Otherwise, you can hide the numbers until needed.

To show or hide control point numbers

1.       Set the Point Numbers option in the Display Options dialog.
