Reviewing ‘Cvt’ Values for Links

Every link has what looks like a highway sign. When you want to review or change the ‘cvt’ value assigned to a link, you should display those signs. Otherwise, hiding them makes the links and other on-screen hinting easier to see.

The X links of a Times New Roman “B” with highway signs showing ‘ cvt’ values.

You can display large highway signs that include the sequential line number of the ‘cvt’ value used by each link, or you can turn on smaller, unnumbered signs. For details, see “Hinting with ‘Cvt’ Values”

Note A distance will not display a highway sign.
