Opening and Closing a Font

Visual TrueType can open any TrueType font whose file type is ‘sfnt.’ Visual TrueType cannot open TrueType Collections nor sfnt files with only CFF data.

Locate the font you want to open.

To open the font you want to hint

1.       On the File menu, click Open, or press Ctrl + O.

2.      In the Open dialog box, locate and click the font file.

3.      Click Open.

Alternatively, you can open a recently opened font by selecting it from the Recent Files list on the File menu.

To close the open font

1.       Select the Main window.

2.      On the File menu, click Close, or press Ctrl + W.

3.      If Visual TrueType prompts you to save your most recent changes, click a button:

·       To close after Visual TrueType saves your changes to the file name shown in the title bar of the main window, click Yes.

·       To save the changes but as a new file, click Cancel and use the Save As command to save and rename the font file before quitting (see “Saving the Font File”).

·       To close without saving your changes, click No.

Note VTT is capable of importing compiled, binary instructions from the font to the low-level language windows. This is useful if the font you are working on has no source data. When you open a font, VTT will ask you whether or not you want to import these instructions. If you do not require this functionality and wish to disable it permanently, select the “Don’t show this message again” in the dialog box. Note: Selecting this option disables this prompt permanently. If you later want to import binary data, you must use the “Source from binary” command on the Tools menu.
