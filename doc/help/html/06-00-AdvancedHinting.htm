Advanced Hinting

The graphical interface in Visual TrueType handles most hinting requirements. However, if you’re an advanced user or otherwise familiar with TrueType tables, you have more options. You can:

·       Make additional changes to the control value table (‘cvt’), as well as the ‘prep and ‘glyf’ tables used by Visual TrueType.

·       Import a glyph from another font to take advantage of hinting already done to it.

·       Do additional hinting in the VTT Talk window by entering text-based commands.
