Changing the Point Size of the Glyph

The main window displays only one glyph at a time. When hinting, you review and adjust the same glyph outline at many sizes, so you will change sizes often before you finish hinting a particular glyph. When you change sizes, the outline shows any hinting already applied to the glyph.

Visual TrueType offers several ways to change the point size of the glyph. You can use the size run, the keyboard, or commands from the Size submenu of the Display menu.

The Size menu lists the most common sizes for hinting. You can use the Other Point Size command to specify any other size between 4 points and 255 points.

 

If the Size menu doesn’t list the size you want,  use the Other Point Size command.

To change the point size of the current glyph

1.       If the size run is displayed, click any number or glyph shown there.

OR

1.       To change size by one point, press the up arrow key (increase) or down arrow key (decrease).

OR

1.       To change size by 10 points, hold down the Shift key, and press the up arrow key (increase) or down arrow key (decrease).

OR

1.       On the Size menu, click any size.

OR

1.       On the Size menu, click Other Point Size (Ctrl + =).

2.      After Point Size, type a new value.

3.      Click Set.
