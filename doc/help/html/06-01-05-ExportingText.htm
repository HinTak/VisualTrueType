Exporting Text

You can export the contents of a text-based window and save it as a text file. Later, you can import that text file into another font or edit it with any text editor or word-processing software, such as WordPad or Microsoft Word.

This is handy, for example, if you want to export the Font Error window and then send that text file as an e-mail attachment to another typographer. Or you may want to save your completed ‘cvt’ table—you could use it with other fonts in the same family, or you could name it CvtTmpl.txt and save it in the Visual TrueType folder as a template.

To export a text-based window

1.       Select the window you want to export by clicking on the window, choosing it in the View menu, or pressing its associated keyboard shortcut.

2.      From the Export submenu of the File menu, click “Text from window”.

3.      Specify where you want to save the new file.

4.     Type the file name for the exported file.

5.      Click Save.
