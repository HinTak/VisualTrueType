Going to a Line

Sometimes you will want to go to a particular line or entry. For example, the Font Error window may give the line number of an error in an OpenType table.

If you know the line number with the data you want to review or modify, this direct approach can be faster than scrolling through a lengthy table or data file.

Type a line number from the selected text-based window.

If the compiler finds an error in the VTT Talk, TrueType, control value, font program, or pre-program window, Visual TrueType displays an error message, then opens the problem window and highlights the line cited in the error message.

To go to a line

1.       On the Edit menu, select the Go To submenu and choose Line, or press Ctrl + L.

2.      Type the line number to go to.

3.      Click OK.
