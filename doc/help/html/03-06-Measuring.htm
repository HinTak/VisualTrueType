Measuring in the Main Window

While hinting, you’ll need to measure various distances on the glyph so you can enter and edit values in the control value table (see “Using the Control Value Table”). To measure distances in the main window, select the Measuring tool or press F9:

The pointer changes to a caliper that you drag to measure:

If you start or stop dragging near a control point or the glyph outline, the pointer will “snap to” the point or outline. This assures accurate measurements between points, even at small views. A heavy black line starts where you begin dragging and continues to where you release the mouse button.

Drag the Measuring tool for precise measurements.

As you drag, the status bar changes, too. The top line shows the glyph index, Unicode number, and unit of measure.

 

The status bar shows how far you’ve measured.

Below that, the status bar updates these values:

The measured line remains until you start measuring again or select another tool. If you point on either end of the line, the other end will stay anchored as you pivot the line or drag it more. The line disappears when you click anywhere else in the window or select another tool.

To measure any distance in the main window

1.       In Display Options, disable the Grid Fit and Pixels options to make measurements in font units. Leave either of these options set to measure distances in pixels.

2.      Select the Measuring tool by either clicking its button in the toolbar or by pressing F9.

3.      Point where you want to start measuring, and drag:

·       If you start or end at a control point, the point turns into a black circle, and the measured line has arrows at each end.

·       If you start or end on the glyph outline, a heavy short line displays on the outline, and the measured line has arrows at each end.

·       If you drag almost perpendicular to the glyph outline, the measured line locks into a perpendicular position, with arrowheads inset from the two ends.
