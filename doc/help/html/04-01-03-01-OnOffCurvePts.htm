Working with On-Curve and Off-Curve Points

Control points can be on or off the glyph outline (“on curve” or “off curve”):

·       The blue “on curve” points form joints in the outline. Moving one of these points creates an angle on the glyph outline, changing the direction of the outline.

·       The white “off curve” points affect the glyph outline less directly, changing the curve of the outline as you drag the point.

At any time, you can “swap” a control point that’s on the curve to off the curve and vice versa. However, to see, add, move, or delete off-curve control points, Show Fewer Points must not be selected on the Display menu (or in the dialog box for the Options command from the Display menu). Otherwise, Visual TrueType shows only on-curve control points.

To work with both on-curve and off-curve points

1.       Set the Show Fewer Points option in the Display Options dialog.

To swap a control point

1.       Set the Show Fewer Points option in the Display Options dialog.

2.      In the toolbar, click the Swap Point tool.

3.      Click the control point you want to swap. The black on-curve point turns white, a white off-curve point turns black.
