Working on a Different Glyph

At any time, you can switch to another glyph in the main window. You may want to hint it, review hinting you’ve already done to the glyph, or take a snapshot to refer to while working on another glyph.

You can switch to another glyph by typing its character with your keyboard or click it in the Character/Glyph Set window, if open.

Otherwise, you can display any glyph by selecting the Go To submenu from the Edit menu, and clicking Character. In the Goto Glyph dialog, you can specify the glyph’s character code or glyph index:

·       The glyph index is the sequence number assigned by the font designer, so it varies from font to font.

·       The character code is the Unicode value, which is the same from font to font.

 

Use the Go To Glyph/Character command to specify the glyph you want to work on next.

Although this approach is slower, its advantage is that you can then click the Changer tool to switch between the last two glyphs you reviewed with the Go To / Glyph command:

Otherwise, you can use commands or the keyboard to proceed sequentially to the next glyph or previous glyph in the character set. Visual TrueType determines the sequence of the glyphs based on whether you selected Glyph Index Based in the dialog box for the Preferences command from the File menu (for details, see “Setting Preferences”).

To change to another glyph

1.       On your keyboard, type the glyph you want.

OR

1.       In the View menu, click Character Set.

2.      In the Character/Glyph Set window, click the glyph you want.

To select a glyph by character code or glyph index

1.       Select the Go To submenu of the Edit menu, and click Go to Glyph/Character. Alternatively, you can press Ctrl + H.

2.      Click Character Code or Glyph Index, based on the system you will use.

3.      In the Which Char box, type the appropriate value.

4.     Click OK.

To switch between the last two glyphs selected with the Go To Glyph/Character command

1.       In the toolbar, click the Changer tool or press F10.

To go to the next glyph in the character set

1.       Click the right arrow key (next glyph or character) or left arrow key (previous glyph or character).

To go to the next glyph number by character code or glyph index

1.       On the Edit menu, select the Go To submenu and click Next Glyph/Character.

To go to the previous glyph number by character code or glyph index

1.       On the Edit menu, select the Go To submenu and click Previous Glyph/Character.
