What’s in the Toolbar

The toolbar on the left edge of the main window contains 24 tools. If your screen cannot display all the tools at one time, you can scroll up or down the toolbar by clicking scroll arrows at its top. The tools fall into four categories:

·       Viewing tools (Panning, Measuring, Zoom-in, Zoom-out)

·       Control point tools (Move Point, Swap Point, Delete Point, Add Point)

·       Hinting tools (Angle, XLink, YLink, XShift, YShift, XInter, YInter, Stroke, Align, XDelta, YDelta, XMove, YMove)

·       Display tools (Camera, Jump Back, and Size Run)

To select a tool, click it or press the associated function key. The mouse cursor will change to indicate the selected tool.

Not all tools will be available at all times. For example, the X tools will be dimmed unless Show X Direction is checked in the Display options. Similarly, the Y tools will be dimmed unless Show Y Direction option is set. For details, see “Setting Options Useful for Hinting”.
