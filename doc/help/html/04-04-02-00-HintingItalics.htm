Hinting Italic Glyphs

Before adding links, interpolations, and other hinting to an italic font, you must do three tasks:

·       Enter the italic size run in the ‘cvt’ table (for the entire font).

·       Set the angle of the stroke (for each glyph).

·       Minimize stair-stepping pixels by changing the rounding method on control points.

The section describes those tasks. When you’ve finished, continue with conventional hinting.
