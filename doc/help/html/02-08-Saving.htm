Saving the Font File

As you work with Visual TrueType, you should periodically save your changes with the Save command on the File menu. The changes you save update internal source data for the open font.

Note Saving the font updates the source data but does not compile the changes in the usable font. To compile the font, see “Compiling Your Results”.

If you want to save the font file with a new name (for example, you are saving different versions or want to make a backup copy), use the Save As command on the File menu.

 

Use the Save As command on the File menu to rename the font file or store it in a new location.

To update an existing file

1.       On the File menu, click Save, or press Ctrl + S.

To save the font as a new file

1.       On the File menu, click Save As.

2.      Specify where you want to save the new file.

3.      In the Save As text box, type a new file name.

4.     Click Save.
