Moving Around in the Window

All Visual TrueType windows have standard Windows scroll bars that you can use to move the current glyph up and down, left and right in the window. You also can use the Home, End, and—with or without the Shift key—Page Up and Page Down keys.

To move in any direction and with more control over the precise distance, use the Navigate tool to drag the glyph as if it were on a sheet of paper behind the window. The Navigate tool looks like a hand:

So does the pointer when you select the tool:

The Navigate tool is so commonly used that it has two keyboard shortcuts (the Shift key, or space bar) for panning while another tool is still selected. When the glyph is where you want it, release the keyboard shortcut to resume work with the previous tool.

To move horizontally or vertically

1.       Use the scroll bars.

OR

1.       Press Home, End, Page Up, Page Down, Shift + Page Up, or Shift + Page Down.

To pan the window

1.       Select the Navigate tool by pressing its button in the toolbar.

2.      Point where you want to start panning, and drag in any direction.

OR

1.       Hold down the Control key or the space bar.

2.      Point where you want to start panning, and drag in any direction.
