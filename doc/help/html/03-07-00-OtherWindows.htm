Working with Other Windows

Although you can hint only one glyph at a time in the main window, you can open two other graphical windows that might be useful for visual hinting: the Character/Glyph Set Window and Snap Shot of Main Window. All three graphical windows are available as commands on the View menu with keyboard shortcuts: Main Window (Ctrl + 1), Character Set (Ctrl + 9), or Snap Shot of Main (Ctrl + 0).

From the View menu, you can open several text-based windows that enable you to edit font data. All users will need to open and edit the Control Program window, which is a text window of source data for the control value table, or ‘cvt’ (see “Using the Control Value Table”). More for advanced users who need access to the technical data behind Visual TrueType. For details about other text-based windows, see “Advanced Hinting.”

Only one window can be active at a time. Use standard Windows techniques to select, position, and size these windows conveniently for you.

Closing the main window also closes the font. For details, see “Opening and Closing a Font”.

To open a window

1.       On the View menu, click the window you want. All windows have a shortcut key of Ctrl pressed with a numeral. See the Keyboard Shortcuts section for a reference.

To close a window

1.       Click its close box.

OR

1.       On the File menu, click Close, or press Ctrl + W.
