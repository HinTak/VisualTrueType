Rounding Control Points

Telling the computer how to round control points seems like a logical first step in hinting. However, because rounding itself seldom produces satisfactory results, Visual TrueType makes it a task you typically do in conjunction with other hinting techniques. Visual TrueType will automatically do rounding when appropriate, such as when you create links.

You can round pixels while using one of the linking (XLink and YLink), interpolating (XInter and YInter), or shift (XShift and YShift) tools. With a hinting tool select, just hold down the right mouse button as you right-click on the head of a link, and Visual TrueType displays a menu of rounding options in the same direction as the tool. To set the rounding type while using hinting tools

1.       Point to the control point you want to round.

2.      Hold down the right mouse button to display a menu of rounding options and select the appropriate option.

3.      Drag to the option you want.
