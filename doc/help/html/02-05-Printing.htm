Printing to Check Hinting Results

Visual TrueType gives you many options for printing hinted glyphs and tables of the technical data used for hinting and by the font itself.

At any time, you can print one glyph, several glyphs, or all glyphs to check the results of the hinting you’ve done so far. For the specified glyph(s), you also can choose to print a copy of the main window, the Visual TrueType Talk source code, and the ‘glyf’ program assembled from that source code.

 

Choose print options for one or more glyphs or for the entire font.

Other printing options apply to the entire font. For example, you can print any of the three tables used by Visual TrueType: the control value (‘cvt’), font program (‘fpgm’), and pre-program (‘prep’) tables. Advanced users may want to print copies of the OpenType tables.

In addition, you can print a glyph palette and Font Error window:

·       The glyph palette shows every glyph in the character set at the same size and on the same baseline. The order of the glyphs is by glyph index number or character number, whichever you specified for Glyph Index Based in the dialog box for the Preferences command (see “Setting Preferences”).

·       The Font Error window displays a cumulative record of all errors the rasterizer detects as you hint the font (see “Using the Font Error Window”).

To save you time, Visual TrueType selects options appropriate for the window you are in when you try to print. For example, if you are in the Control Value Table window, Visual TrueType selects the option for that table. Use those default settings or select others you want.

To print your hinting results

1.       On the File menu, click Print (Ctrl + P).

2.      To print glyph results, click one option to select the glyph(s):

Option

What It Means

This Glyph

Selects only the glyph in the main window.

These Glyphs

Selects all glyphs whose glyph index number you type in the text box. Use a hyphen to indicate a range of glyphs and a comma to separate individual glyphs and ranges.

All Glyphs

Selects every glyph in the font.

Note This option requires significantly more time and paper for printing whatever you select in Step 3.

3.      If you selected glyph(s), click one or more check boxes after For Each Glyph, Print:

Option

What It Means

Main Window

Prints a page showing the glyph as it would appear in the main window.
Note Visual TrueType defaults to this when you select any glyph option.

Visual TrueType Talk

Prints the source code for the glyph.

TrueType (‘Glyf’) Program

Prints the TrueType code assembled from VTT Talk.

4.     To print font tables or other data for the entire font, click one or more options in the “For this font, print” section.

5.      Click Print.

6.     In the Printer dialog box, click to select all options you want.

Note If you have been hinting for grayscale and your printer has a Color/Grayscale option, select it.

7.      Click Print.
